# Pict-Section-Form

> A dynamic, configuration-driven forms framework built on the Pict application platform

- Declarative form configuration through JSON manifests
- Two-way data binding with automatic UI sync
- Expression solvers for calculated fields and conditional logic
- Extensible input types and flexible templates
